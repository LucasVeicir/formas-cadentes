# formas cadentes

projeto para fins educativos simulando a queda de 3 corpos com propriedades e formas diferentes.

---
## 📚 Aprendizados
- motores de física


## ⚙️ Funcionalidades
- 3 corpos caem simultaneamente

## 🛠️ Stack
-  p5
- JavaScript
- HTML5
- CSS3
-  matter.js

## 🚀 Deploy
1. Clone ou baixe este repositório.

```bash
git clone https://github.com/LucasVeicir/formas-cadentes
```

2. Abra o arquivo `index.html` em seu navegador.
